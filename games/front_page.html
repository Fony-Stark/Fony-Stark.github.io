2048

The game 2048, is a game which is based on the exponentiels of twos. You have a 5x5 board, each time you spend a turn there will spawn a new, either 2 or 4 element on the board. Each turn has 4 options, LEFT, RIGHT, TOP or DOWN. These refere to which way you want the blocks to move. If two elements with the same value collides they will intervine and become their product after addition.
